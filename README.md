# Prompt Gen GUI

インタラクティブなキーワード管理ツールです。キーワードをクリックしてプロンプトを素早く作成・コピーしたり、キーワードリストを JSON で保存/読み込みできます。

## インストール

開発環境で動かす場合:

```bash
python -m pip install -e .
```

## 実行

- ソースから: `python -m prompt_gen_gui` または `prompt-gen-gui`（インストール後）
- Windows exe を作るには PyInstaller を使い、`build/version.txt` を `--version-file` に指定してください。
