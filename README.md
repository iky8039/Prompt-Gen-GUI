# Prompt Gen GUI

このプロジェクトは、AIへのプロンプト作成を効率化するためのインタラクティブなキーワード管理ツールです。特定のキーワードを組み合わせて素早くプロンプトを構成し、クリップボードにコピーしたり、キーワードリストをJSON形式で保存・管理したりすることができます。

## 🚀 リリース情報
- **最新版 (.exe)**: [Releases](https://github.com/YOUR_USERNAME/YOUR_REPO/releases) から実行ファイルをダウンロードして、ローカルPCで直接利用できます。

## ✨ 主な機能

- **プロンプト構築**: キーワードをクリックするだけでカンマ区切りのプロンプトを作成。
- **キーワード管理**: キーワードの追加、検索、削除（Delete Mode）が可能。
- **インポート/エクスポート**: キーワードリストを `.json` ファイルとして保存・読み込み。
- **マルチデバイス対応**:
    - **Desktop (.exe)**: Tkinterを使用した高速なデスクトップアプリ。
    - **Colab/Notebook**: `ipywidgets` を使用したブラウザ完結型ツール。
    - **Mobile**: スマートフォンからの操作に最適化したUI。

## 🛠 使い方

### ローカル実行 (.exe)
1. Releaseページから `prompt_builder.exe` をダウンロードします。
2. ダブルクリックして起動します（初回起動時にセキュリティ警告が出る場合は「詳細情報」→「実行」を選択してください）。

### Google Colab での利用
リポジトリ内の `.ipynb` ファイルを開き、対応するセル（PC版またはMobile版）を実行してください。

## 📂 キーワードリストの形式
キーワードは以下のような単純なJSON配列で管理されています。
```json
[
    "AI",
    "Python",
    "Data Science"
]
```
