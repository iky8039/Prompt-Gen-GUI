# Prompt Gen GUI v1.5 Release Note (ドラフト)

## 主な新機能・変更点
- 多言語統合エイリアス辞書（日本語・英語・中国語など）に対応
- 検索欄・プロンプト欄でShift+Enterを押すと、
  - 入力内容とエイリアス辞書との差分を自動判定
  - 未登録ワードのみをキーワードリストに一括追加
- 大規模辞書でも高速検索・追加が可能
- 既存のUI・操作性はそのまま

## 使い方
- 検索欄またはプロンプト欄に複数ワードを入力
- Shift+Enterで未登録ワードのみ自動追加
- 多言語（日本語・英語・中国語）での検索・追加が可能

## 辞書ファイル
- py/keyword_aliases.json（統合エイリアス辞書）を同梱
- 必要に応じて差し替え・拡張可能

## 注意事項
- 大量の新規ワード追加時は一度に追加されます
- 旧バージョンの辞書・設定ファイルは上書きされません
