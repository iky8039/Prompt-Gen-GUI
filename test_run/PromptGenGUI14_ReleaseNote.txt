# Prompt Gen GUI v1.4 リリースノート

## 主な変更点
- エイリアス（多対1）対応：
  - 例："apple" で「りんご」「林檎」などの別名でも検索可能
  - R18姿势・nai tag等の大規模辞書jsonをサポート
- エイリアスjsonの全英語キーワードを初期リストに自動追加
- UI・動作の安定性向上

## 使い方
1. `PromptGenGUI14.exe` を実行
2. `py/keyword_aliases.json` を同じフォルダ構成で配置
3. 検索欄に中国語・日本語・英語いずれかの別名を入力
4. 対応する英語キーワードがヒット

## フォルダ構成例
```
PromptGenGUI14.exe
py/
  keyword_aliases.json
```

## 備考
- 旧バージョンからのアップグレード時は、py/keyword_aliases.jsonを上書きしてください。
- 辞書jsonの内容は自由に拡張可能です。
